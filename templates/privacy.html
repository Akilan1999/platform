Privacy Policy

Information We Collect

To facilitate the ability of users to vote, we create a unique 'voter ID' which is generated using the following information:

  • IP address
  • User-agent header
  • Random number generated at that instance (pepper). This is not linked to the session ID
A cryptographic hash is created using this information in order to roughly identify a user, mitigating potential abuses of the anonymity granted through the service.

On the complaints page users may leave an email address if further correspondence is required.

Cookies

Session cookies ("hithereimacookie") are used to generate a temporary username, allowing a user to comment anonymously and create discussions. This session ID expires every 24 hours.

Examples of a session account:

Grateful Golang

Session ID: c2c1390516e18d14

We use Cloudflare to provide enhanced security to our service, thus you may notice another cookie called __cfduid which is their anonymised cookie. This does not, however, track you between sites.

Third Parties

There's only one party happening here. No information is shared with third parties.

Changes

We, the current class representatives, reserve the right to make any changes to this privacy policy. Notice will be given as an alert on the homepage.
