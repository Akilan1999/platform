Information We Collect

To facilitate the ability of users to vote and create discussions, we create a unique 'voter ID' which is generated using the following information:

  • IP address
  • User-agent header
  • Random number generated at that instance (pepper). This is not linked to the session ID
A cryptographic hash is created using this information in order to roughly identify a user, mitigating potential abuses of the anonymity granted through the service.

Cookies

Session cookies are used to generate a temporary username, allowing a user to comment anonymously. This session ID expires every 24 hours. No other information is stored in the browser.
